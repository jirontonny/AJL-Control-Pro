# AJL Control Pro

## Sistema Administrativo para Agente Banpro

Versión: 1.0 (En desarrollo)

---

# Objetivo

Desarrollar un sistema profesional para el control de operaciones de un Agente Banpro utilizando Microsoft Excel LTSC 2021 con VBA.

---

# Módulos

- Dashboard Ejecutivo
- Movimientos
- Caja Banpro Córdobas
- Caja Banpro Dólares
- Control de Préstamos
- Arqueo
- Reportes
- Usuarios
- Bitácora
- Configuración

---

# Tipos de Operaciones

- Depósito
- Retiro
- Pago de Servicio
- Remesa
- Préstamo a Tienda
- Devolución de Préstamo

---

# Monedas

- C$
- USD

---

# Flujo del sistema

Dashboard

↓

Nuevo Movimiento

↓

Validación

↓

Guardar Movimiento

↓

Actualizar Caja

↓

Actualizar Dashboard

↓

Actualizar Reportes
